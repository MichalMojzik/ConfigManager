Detailed Description

Trida spravujici soubory s nastavenim. Vzhledem k tomu, ze soubory maji format ".ini" a jejich format tedy neni uplne specifikovany, uchovava tato trida puvodni retezcove hodnoty. Umoznuje pristupovat k sekcim a hodnotam daneho souboru pomoci metody SpecifySection. V destruktoru jsou pripadne zmeny zapsany zpet do souboru.
